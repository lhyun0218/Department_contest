## 🏠 AI 전세사기 예방 서비스 (안심 전세 스코어)

---

### ✨ 소개 (Introduction)

이 프로젝트는 증가하는 **전세 사기**의 심각성을 인지하고, 기존의 사후 복구 중심 대책의 한계를 극복하기 위해 기획되었습니다. **AI 기술**을 활용하여 계약 전 주택 개별의 **위험도(안심 전세 스코어)**를 예측하고, 이를 통해 부산 시민의 재산 피해를 **선제적으로 예방**하는 것을 목표로 합니다.

* **팀명:** 지역사회 S.O.S (Society Of Safe; 안전한 사회)
* **프로그램:** 2025 AI 학과특화 프로그램 (전학년 학과 대회)
* **핵심 목표:** 주택 개별의 위험도를 측정하고 임대인의 패턴을 파악하여 사전에 피해 발생을 차단합니다.

---

### 🚀 주요 기능 및 핵심 기술 (Features & Core Technology)

#### 1. 안심 전세 스코어 측정

주택 관련 데이터를 분석하여 해당 주택의 전세 계약 위험도를 **'높음', '보통', '낮음'**의 3단계로 분류하고 스코어를 제공합니다.

* **위험 등급 및 텍스트:**
    * **높음 (High):** 주의 필요
    * **보통 (Medium):** 추가 확인
    * **낮음 (Low):** 안전

#### 2. 웹 인터페이스 및 지도 연동

사용자가 직관적으로 주택 위험도를 확인할 수 있도록 웹 인터페이스를 구현했으며, **Leaflet.js** 라이브러리를 이용해 지도 상에 주택 위치와 위험도를 표시했습니다.

* **사용 기술:** HTML, CSS, JavaScript (Leaflet)
* **표시 정보:** 가격, 면적, 유형, 건축 연도, 등기 여부, 근저당 설정 금액 등 부동산 정보 및 위험 등급

#### 3. AI 모델 기반 분석

* **KoBERT 모델 활용:** SKT에서 개발한 **KoBERT(Korean BERT)** 모델을 활용하여 전세 사기 관련 데이터를 분석하고 위험도를 예측하는 데 사용했습니다.

### 🙏 기대 효과 (Expected Outcomes)

* **사전 예방 강화:** 사후 복구 중심이 아닌, 계약 전 위험도 확인을 통한 선제적 예방 효과.
